利用可能なエンドポイント

GET / - このドキュメントページ
GET /test-chat.html - チャットAPIテストページ
GET /api/health - ヘルスチェック
GET POST /api/chat - チャット機能（GETで仕様表示、POSTで実行）
POST /api/completion - テキスト生成

環境設定

ローカル開発では以下の環境変数が必要です：

# .env ファイルに設定 DIFY_API_KEY=your_dify_api_key_here DIFY_BASE_URL=https://api.dify.ai

注意: 実際のDIFY_API_KEYを設定しないとAPIは動作しません。

curlでのテスト例

# ヘルスチェック curl http://localhost:3000/api/health # チャット curl -X POST http://localhost:3000/api/chat \ -H "Content-Type: application/json" \ -d '{"message": "こんにちは"}'
