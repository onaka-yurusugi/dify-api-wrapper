🤖 Dify API Wrapper

DifyのチャットボットAPIをラップするサーバーレスAPIです。TypeScriptで型安全性を提供し、フロントエンドから簡単に利用できます。

📚 API エンドポイント

POST /api/chat

Difyチャットボットとの会話を行います。

パラメータ: message, conversation_id (オプション), user_id (オプション)

POST /api/completion

Difyを使用してテキスト生成を行います。

パラメータ: inputs, user_id (オプション), response_mode (オプション)

GET /api/health

APIの稼働状況と設定を確認します。

GET /api/

詳細なAPIドキュメントとサンプルコードを表示します。

🚀 使用方法

詳細なAPIドキュメントとサンプルコードは /api/ エンドポイントで確認できます。

✨ 特徴

  • ✅ TypeScriptで型安全性を提供
  • ✅ CORS対応でフロントエンドから直接利用可能
  • ✅ 包括的なエラーハンドリング
  • ✅ 環境変数による設定管理
  • ✅ Vercelでの高速デプロイ
